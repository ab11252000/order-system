⚙️ 首次設定：請先完成下方「JSONBin 雲端設定」，才能讓公告同步到 GitHub Pages 上的首頁。
☁️ JSONBin 雲端設定 只需設定一次，之後每次儲存都會同步到線上
1
註冊 JSONBin 帳號
前往 jsonbin.io 免費註冊，完成後進入後台。
2
取得 Master Key
登入後點右上角帳號 → API Keys，複製 Master Key（以 $ 開頭的那串），貼到下方。
3
建立公告資料庫
輸入 Master Key 後，點下方按鈕自動建立一個雲端資料庫（Bin）。
4
把 Bin ID 填入首頁
複製下方的 Bin ID，打開 首頁.html，找到這行並替換：
var BIN_ID = 'YOUR_BIN_ID';
